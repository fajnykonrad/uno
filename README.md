Es necessari instalar la dependencia rich. (pip install rich)

Per iniciar el servidor cal anar a la carpeta uno y executar "python -m server.main" a la terminal
Per jugar com a usuari cal anar a la carpeta uno y executar "python -m client.main" a la terminal.

Es possible que per sistemes Mac o Linux n'hi hagin dificultats.
